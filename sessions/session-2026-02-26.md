## What I understood
- Some operations apply to each row and give a result to each row
- AVG, SUM, COUNT, MAX, MIN (complete colums) returns a single value
-  Group by subdivides a table by the column i tell it to 
- SELECT category, AVG(price) from productos GROUP BY category  returns the average price for each category 

- HAVING is applied after the WHERE

- Order:
SELECT
FROM
WHERE
GROUP BY
HAVING
