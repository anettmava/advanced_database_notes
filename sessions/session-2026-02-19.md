## What I understood
- If tables have a column with a common row we can merge them using JOIN
- SELECT * FROM students JOIN cursos ON student.curso_id = cursos.id
- INNER JOIN are the records that are in both tables
- LEFT JOIN : the elements on the left table and the matched on the right side table
- RIGHT JOIN : the elements on the right table and the matched on the left side table
- CROSS JOIN: each of the products at the left with all the products at the right
- SELF JOIN: use an alias to join a table to itself 
SELECT e1.nombre AS empleado, e2.nombre AS supervisor
FROM empleados AS e1
JOIN empleados AS e2 ON e1.id_supervisor = e2.id_empleado;

This querie returns a list that pairs each name with its supervisor using the SELF JOIN to compare the relationships inside the same table
